# game-of-life
ncurses implementation of Conway's Game of Life

```
HELP

trace (t)
circle (o)
line (l)
rectangle (r)
copy (d)
paste (p)
clear (c)
save to buffer (s)
load from buffer (0-9)
next frame (enter)
quit (q)
toggle sidebar (?)
```
